# lab1/python

This directory contains the Python project template for Lab 1.

For a description of the lab assignment, see the Canvas page.

## Installing Python

The recommended way to install Python is through your OS's package manager.
We recommend using Python 3.11 or newer.

## Installing dependencies

This lab requires the Python package `pytest`, which you can install either by running `pip install -r requirements.txt` in this directory, or through your OS's package manager. Depending on the package manager, it might be called `pytest` or `python3-pytest`.
